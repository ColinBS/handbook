Usage of the RTRlib
===================

Build and Install RTRlib
------------------------

Usage Examples
--------------

Run the ``cli-validator`` tool as follows:

.. code-block:: Bash

    cli-validator rpki-validator.realmv6.org 8282

Run the ``rtrclient`` tool as follows:

.. code-block:: Bash

    rtrclient tcp -k -p rpki-validator.realmv6.org 8282
